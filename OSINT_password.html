💡 왜 위험한가요?
스마트폰으로 찍은 사진에는 EXIF 메타데이터가 포함됩니다. 여기에는 GPS 좌표(촬영 위치), 촬영 시간, 카메라 기종, 소프트웨어 버전 등이 기록되어 있어요. 이 정보만으로 해커는 당신의 집 위치, 출퇴근 경로, 사용 기기, 생활 패턴까지 파악할 수 있습니다.
방어법: 사진 업로드 전 메타데이터 제거하기, SNS 업로드 시 위치 정보 끄기, 아이폰은 설정 → 개인정보 → 위치 서비스 → 카메라 → '안 함'으로 설정

💡 안전한 비밀번호 만들기
1. 12자 이상으로 만들기 (길이가 가장 중요!)
2. 대문자 + 소문자 + 숫자 + 특수문자 조합
3. 사이트마다 다른 비밀번호 사용
4. 개인정보(생일, 이름, 전화번호) 절대 포함 금지
5. 문장형 비밀번호 추천: "나는2025년에커피3잔!" (기억하기 쉽고 안전함)
